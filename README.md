# Basic Setup
1. Install or upgrade pip
    ```
    python.exe -m pip install --upgrade --user pip
    ```
2. Python installation
    - Verified python versions
        ```
        python --version
        ```
    - Install python
        ```
        pip install python
        ```
3. Flask installation
    ```
    pip instal flask
    ```
